이 가이드는 Python(Django) 개발자가 Kotlin + Spring Boot + Gradle 기술 스택을 처음부터 이해할 수 있도록 작성되었습니다. "k도 모르고 s도 모르고 g도 모른다"는 전제에서 출발하며, 가능한 모든 개념을 Python/Django와 비교하여 설명합니다.

gate-futures-bot은 Gate.io 거래소의 선물(Futures) 거래를 자동화하는 봇입니다. 아래는 전체 구조를 보여주는 다이어그램입니다.

핵심 요약

이 프로젝트는 Spring Boot로 애플리케이션을 구성하고, Kotlin Coroutines으로 비동기 처리를 하며, Gate.io Java SDK를 통해 거래소 API를 호출합니다. application.yml에서 API 키, 거래 대상, 레버리지 등을 설정합니다.
