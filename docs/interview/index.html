Python → Kotlin/Spring 전환 가이드

Django/DRF 개발자가 Kotlin + Spring Boot 3년차 수준에 도달하기 위한 집중 학습 자료

티니어 백엔드 지원 맞춤 Python/Django/DRF 경력자 대상 사이드 프로젝트 기반

이 가이드는 누구를 위한 것인가

Python(Django, DRF)으로 백엔드를 만들던 개발자가 Kotlin + Spring (Boot) 스택으로 옮겨 3년차 수준의 기술 이해를 갖추는 것이 목표입니다. 단순히 "문법 번역"이 아니라, 왜 그렇게 쓰는지(관용구, 철학), 어떻게 운영하는지(테스트/관측성/배포)를 설명합니다.

사이드 프로젝트와 함께 보기

이 저장소의 gate-futures-bot은 Gate.io 선물 자동매매 Kotlin + Spring Boot 봇입니다. 각 개념이 실제로 어떻게 쓰였는지 코드를 직접 볼 수 있습니다. 이 가이드는 그 프로젝트를 면접에서 어떻게 설명할지까지 포함합니다.
